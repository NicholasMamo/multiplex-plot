Text¶

Text Annotation¶

A class of visualization that allows text annotations. The annotation class is mainly concerned with organizing text.

class text.annotation.TextAnnotation(drawable)¶

A class of visualization that allows text annotations. The text.annotation.TextAnnotation is mainly concered with organizing text.

Variables:drawable (drawable.Drawable) – The axis where the text annotation visualization will be drawn.
__init__(drawable)¶

Initialize the text annotation with the figure and axis. The figure is used to get the renderer. The visualization is drawn on the given axis.

Parameters:drawable (drawable.Drawable) – The axis where the text annotation visualization will be drawn.

list of weak references to the object (if defined)

_align(tokens, line, wordspacing, linespacing, align='left', *args, **kwargs)¶

Organize the line tokens. This function is used when the line overflows.

Parameters:
  • tokens (list of matplotlib.text.Text) – The list of tokens added to the line.
  • line (int) – The line number of the tokens.
  • wordspacing (float) – The space between words.
  • linespacing (float) – The space between lines.
  • align (str) –

    The text’s alignment. Possible values:

    • left
    • right
    • justify
