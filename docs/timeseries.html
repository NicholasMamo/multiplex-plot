Time Series¶

Time series are among the most common types of visualizations, and with good reason. They are easy to create because they are simple, which means that they are also intuitive. All too often, we rely on that simplicity to tell a story. Multiplex’s time series makes it easier to create time series that tell a story clearly.

Keep reading to learn more about time series visualizations.

Time Series¶

The timeseries.timeseries.TimeSeries class borrows heavily on matplotlib’s plot function. This class builds on matplotlib’s plotting and introduces more functionality.

To start creating time series visualizations, create a timeseries.timeseries.TimeSeries instance and call the timeseries.timeseries.TimeSeries.draw() method. If you are using the drawable.Drawable class, just call the drawable.Drawable.draw_time_series() method on a drawable.Drawable instance instead.

draw(*args, **kwargs)¶

Draw a time series on the drawable.Drawable. The arguments and keyword arguments are passed on to the matplotlib.pyplot.plot() method. Thus, all of the arguments and keyword arguments accepted by it are also accepted by this function.
